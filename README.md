## Scurta descriere
Aplicatie realizata la concursul "Severin Bumbaru" editia 2026 alaturi de doua colege. Cerinta a fost sa se implementeze o aplicatie web care sa ajute primaria locala sa-si gestioneze sesizarile.
## Tehnologii utilizate
-Java
-HTML+CSS
-ORACLE SQL
-Spring Boot
-Google Maps API
## Structura Proiectului
  src
    main
      java/com/galati/sesizari (unde se afla toate clasele necesare functionarii logice a aplicatiei)
      resources (unde se afla paginil HTML si alte fisiere pentru partea vizuala)
    test/java/com/galati/sesizari (unde se afla clasa care ruleaza programul prin spring boot)
## Rolul meu in dezvoltarea aplicatiei
Am lucrat impreuna cu o colega pentru partea de backend, am asigurat un server online pentru baza de date pentru ca testarile sa functioneze la fiecare coechipier.
## To do 
Configurarea rolului pentru admin si institutiile care ar trebuii sa gestioneze sesizarile.
