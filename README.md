
For this project, I was responsible for developing the backend architecture using the **Spring Boot** framework for Java. My main contributions include:

* **Layered Architecture Implementation:** Successfully implemented a clean, architecture divided into **Controller - Service - Repository** layers, ensuring proper separation of concerns and business logic isolation.
* **Database Mapping:** Mapped the database's entities and established relationships between models using **Spring Data JPA** and **Hibernate**.
* **REST API Development:** Created the REST controllers (`@RestController`) and implemented fundamental HTTP routes such as **GET** and **POST** to enable seamless communication with the frontend.
* **Data Persistence & Configuration:** Managed the database connection within the Java framework and configured the application's connection pool inside the `application.properties` file for optimal performance.
